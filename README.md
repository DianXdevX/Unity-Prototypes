## 📝 Sobre o Repositório
<img width="1284" height="679" alt="image" src="https://github.com/user-attachments/assets/ac1e353b-1229-48bf-a8c5-b19c2d7ad39c" />

Este repositório serve como um portfólio de evolução e aprendizado prático em desenvolvimento de jogos. 

Originalmente, o projeto nasceu para registrar meu progresso na trilha **Junior Programmer** (Junior Pro-gamer). Hoje, ele funciona como meu laboratório pessoal para testar ideias de mecânicas, treinar lógica em C#, experimentar com assets e arquivar pequenos projetos conceituais.

---

## 📚 O Que Você Encontrará Aqui?

*   **🧪 Testes & Protótipos:** Pequenos experimentos focados em mecânicas específicas (como movimentação, física, colisões, sistemas de pontuação e triggers).
*   **🏗️ Ideias e Projetos Inacabados:** Conceitos de mini-games que comecei para fixar conteúdos estudados.
*   **🧠 Aprendizado Contínuo:** Cada componente representa um degrau importante na minha exploração da engine e no domínio do controle de versão com Git e GitHub.

---

## 📂 Estrutura do Projeto Unity
Todos os backups completos dos arquivos e jogos finalizados de cada fase estão guardados e organizados na aba **[Releases](https://github.com/DianXdevX/Unity-Prototypes/releases)** do repositório.

🕹️ **Jogue no Navegador:** Vários dos protótipos desenvolvidos podem ser jogados e visualizados diretamente no meu perfil oficial do **[Unity Play](https://play.unity.com/pt/user/730ab507-470c-4cd7-8a31-337831dc9a59)**. Venha testar e conferir as builds online!
